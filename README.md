## 🚀 How to Compile and Run

```bash
# General
javac FileName.java
java ClassName

# For packages (Experiment 4)
javac -d . PackageName/ClassName.java
javac MainFile.java
java MainFile

# For Networking (Experiment 12)
# Run server first in one terminal:
java NetworkingDemo server
# Run client in another terminal:
java NetworkingDemo client
```
